Mankind Games Co., Ltd.

I worked here for around three years, during which I contributed to a wide range of projects, including AR applications, social apps, and games. My responsibilities covered
programming, technical art, and also serving as a project leader.

ぱちょふぁ そまちょぱ そまちょぱそ


This game is the very first mobile title based on the popular YouTube animation channel “Hajimemashite Matsuo desu”.

The game’s title comes from a famous nonsensical phrase in “Hajimemashite Matsuo desu.” For convenience, we refer to it simply as pachofa.

It is designed as a casual idle game where players can collect mushrooms during short play sessions and gradually evolve stars and civilizations. Every day, a new chick can also be obtained.

I was involved from the very beginning—taking part in the planning phase, building the program architecture, implementing gameplay systems, and even contributing to animation production.

(Here ↑ is the official game trailer)

In this project, I applied an MVC architecture design and implemented several custom UI features that I developed myself. Some of these features have been made publicly available in

this Git repository  

Technologies Used:

  • Engine & Language: Unity, C#
  • 3D & Animation: Spine, Unity Animation
  • Version Control: Git

結KYOTO


This app is a metaverse application that offers users the opportunity to virtually explore Kyoto together online.

For this project, I developed the avatar system (movement, customization, and animations), the online system (synchronization and communication), as well as the core project architecture.

Technologies Used:

  • Engine & Language: Unity, C#
  • Graphics: Shader Graph, Particle System
  • 3D & Animation: Unity Animation
  • Online: Photon, Vivox
  • Version Control: Git

Mirror Muse


“Mirror Muse” is an app that lets users create a 3D avatar from a single photo and transform it into their ideal self. Avatars can be customized with different outfits, hairstyles, poses, and decorations, making it easy to share unique creations on social media.

In this project, I was responsible for a wide range of features, including the outfit-changing system, tutorial flow, gacha mechanics, mini-games, and the communication lobby. Beyond feature development, I also took on project management and worked as the technical lead. Importantly, I contributed to both the client-side and server-side development, overseeing the full technical scope of the project.

Technologies Used:

  • Engine & Language: Unity, C#
  • Graphics: Shader Graph
  • 3D & Animation: Blender, Unity Animation
  • Cloud: Firebase (Realtime DB, Functions…)
  • Version Control: Git
